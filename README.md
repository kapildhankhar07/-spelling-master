# Spelling Master

Spelling Master is a modern web application designed to help users improve their English spelling skills through daily practice and interactive learning.

## Features

* 📚 Learn from 3000 commonly used English words
* 🇮🇳 Hindi meanings for better understanding
* 🔍 Search words instantly
* 📝 Practice spelling with interactive exercises
* 📱 Mobile-friendly and responsive design
* ⚡ Fast and lightweight web application
* 🎯 Focus on real-world vocabulary used in daily conversations

## Data Source

The application uses the Oxford 3000 word list, which contains some of the most important and frequently used words in the English language.

A custom Python script was developed to convert the word database into JSON format and generate Hindi meanings for efficient use within the application.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* JSON

## Why Spelling Master?

Many learners struggle with spelling and vocabulary retention. Spelling Master provides a simple and effective way to practice frequently used English words and improve communication skills.
